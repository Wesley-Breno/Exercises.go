In a long jump competition, each athlete has five jumps. The athlete's result will be determined by the average of the five remaining values. You must write a program that receives the name and the five distances achieved by the athlete in their jumps and then displays the name, the jumps, and the average of the jumps. The program should terminate when the athlete's name is not entered. The program output should be as in the example below:

Athlete: Rodrigo Curvêllo

First Jump: 6.5 m
Second Jump: 6.1 m
Third Jump: 6.2 m
Fourth Jump: 5.4 m
Fifth Jump: 5.3 m

Final Result:

Athlete: Rodrigo Curvêllo
Jumps: 6.5 - 6.1 - 6.2 - 5.4 - 5.3
Average of jumps: 5.9 m
